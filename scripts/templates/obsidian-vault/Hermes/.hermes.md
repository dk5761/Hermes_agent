## Operating Rules

1. Read `ME.md` + `AGENT.md` before any task
2. Check `wiki/` before searching raw notes
3. Write durable knowledge to `wiki/`, not just chat
4. Raw intake goes to `raw/inbox/` — never modify after deposit
5. Log all wiki changes to `wiki/log.md`
6. Update `wiki/index.md` when adding wiki pages

## Ingest Workflow

When processing items from `raw/inbox/`:

1. Read the raw item
2. Decide: does it deserve a wiki page? (not everything does)
3. If yes: create page in appropriate `wiki/` subfolder
4. Link back to source with `[[raw/inbox/filename]]`
5. Update `wiki/index.md` with new entry
6. Append to `wiki/log.md`
7. Move processed item from `raw/inbox/` to appropriate `raw/` subfolder
